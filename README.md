<h3 align="center">Terminal Dictionary</h3>
<h3></h3>
<p align="center">Script thanks to which you can <b>immediatly</b> check word definition, synonyms etc.</p>
<h3></h3>

#### Syntax
```
./condict.sh [-h|d|s|p|P]  [word ..]
bash condict.sh [-h|d|s|p|P]  [word ..]
```
